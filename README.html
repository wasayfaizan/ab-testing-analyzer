üìä Analysis Modes

Standard A/B Test

Compare two groups (control vs treatment) with binary or numeric metrics. Includes Bayesian analysis, CUPED adjustment, and comprehensive statistical inference.

Multi-Variant Testing

Test multiple variants (A/B/C/D) simultaneously. Performs pairwise comparisons and overall Chi-square/Kruskal-Wallis tests.

Segmentation Analysis

Analyze treatment effects across user segments (demographics, cohorts, etc.). Includes forest plots, insights summary, and targeting recommendations.

Time-Based Analysis

Discover patterns by day-of-week, hour-of-day, or week-of-year. Identify optimal timing for campaigns.

Regression Analysis

Control for multiple covariates simultaneously. Uses statsmodels for full statistical inference with coefficients, p-values, confidence intervals, and odds ratios.

Non-Parametric Tests

Mann-Whitney U and Kruskal-Wallis tests for non-normal data distributions. Perfect for skewed metrics.

üéØ Special Features

Day-of-Week Insights

Automatically extracts and visualizes day-of-week effects from regression analysis. Provides ranking charts, interpretation text, and practical targeting recommendations based on which days show highest conversion likelihood.

Segmentation Forest Plot

Beautiful horizontal bar chart showing treatment lift by segment with 95% confidence intervals. Color-coded by statistical significance for quick visual assessment.

AI-Powered Interpretation

Get natural language explanations of your results, making complex statistical findings accessible to non-technical stakeholders.

Export & Save

Export results as PDF or save experiments for later review. All data is processed locally for privacy.

üìù CSV Format

Your CSV file should include:

  • group - Experiment group (A = control, B = treatment)
  • conversion - Binary metric (0 = no conversion, 1 = converted)
  • revenue - Numeric metric (optional, for revenue-based analysis)
  • Additional columns for segmentation, covariates, timestamps, etc.
group,conversion,revenue,segment,timestamp A,1,25.50,Monday,2024-01-15 10:30:00 A,0,0.00,Tuesday,2024-01-16 11:15:00 B,1,28.00,Monday,2024-01-15 10:35:00 B,1,32.50,Tuesday,2024-01-16 11:20:00
